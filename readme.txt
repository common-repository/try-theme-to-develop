This plugin detect if current user have the capacity of "edit_themes" and then he can change the template to develop it. 

== Description ==

If you have the capacity of "edit_themes" you can see a Try Theme, a new option in Design menu.

In this page you see a form that let you choose a theme and it will be activated while you're login in.

== Installation ==

1. Upload `try-theme-to-develop` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Design, if you have a new optione named `Try theme` it's work.
